Introdução

Bem-vindo à nossa revista interativa sobre tags HTML! Aqui, explicaremos como usar cada uma das tags básicas e algumas avançadas para criar um conteúdo web rico e estruturado.

Títulos: <h1> até <h6>

Parágrafos e Formatação

O texto é estruturado em parágrafos usando a tag <p>.

Exemplo de uso: este texto está destacado, este texto está menor, este texto está em itálico, este está sublinhado e este está em negrito.

Listas Ordenadas e Não Ordenadas

Podemos organizar itens com as tags <ol> (ordenada) e <ul> (não ordenada).

Lista Ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada:

  • Item A
  • Item B
  • Item C

Links e Citações

Os links são criados com a tag <a>:

Citações são representadas por <blockquote>:
